Multiple Traffic Light Simulation System

Project Description
This project is a Smart Intersection Traffic Light Simulation System developed using VB.NET and Windows Forms in Visual Studio. It demonstrates event-driven programming by simulating traffic signal operations, pedestrian crossing requests, and emergency vehicle priority control.

Features
- Automatic North-South and East-West traffic light sequence
- Pedestrian crossing request system
- Emergency vehicle override mode
- Traffic signal countdown timer
- Start, Stop, and Reset controls
- Visual traffic light indicators

 Technologies Used
- Visual Basic .NET (VB.NET)
- Windows Forms
- Visual Studio

  Event-Driven Components
The project uses:
- Button Click Events
- Timer Tick Events
- Label and Control Updates
- Conditional State Transitions

How to Run
1. Open the solution file (`Traffic light.sln`) in Visual Studio.
2. Build the project.
3. Run the application.
4. Use the controls to simulate traffic flow, pedestrian requests, and emergency situations.
